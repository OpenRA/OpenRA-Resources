Share your Maps!

Welcome to the OpenRA map database platform.

OpenRA is a free and open-source RTS game engine that lets you play re-balanced games like C&C: Red Alert, C&C: Tiberian Dawn, Dune 2000...

The engine allows players to create new mods, add modified versions of units, buildings etc. It also offers advanced features for creating maps and missions.

This website makes distribution of OpenRA maps more comfortable.

If you created a new map and want to play it with other players, you just have to:

  • Upload the map here.
  • Create a new game or join any public server and select your map.
  • Other players joining can download the map in-game directly from this website.
